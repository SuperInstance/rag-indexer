┌──────────────────────────────────────────────────────────────────────────────┐
│ CORE PHILOSOPHY                                                                 │
│                                                                                │
│   "Retrieval augments generation with relevant context"                        │
│                                                                                │
│   Timeless Principle: Information Retrieval (Precision & Recall)               │
│                                                                                │
│   precision = |relevant ∩ retrieved| / |retrieved|                            │
│   recall    = |relevant ∩ retrieved| / |relevant|                             │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ CORE ABSTRACTIONS                                                               │
│                                                                                │
│   ┌─────────────┐    ┌─────────────┐    ┌─────────────┐                      │
│   │   Indexer   │    │   Chunker   │    │  Retriever  │                      │
│   │             │    │             │    │             │                      │
│   │ • index()   │    │ • Fixed     │    │ • Dense     │                      │
│   │ • search()  │    │ • Semantic  │    │ • Hybrid    │                      │
│   │ • update()  │    │ • Hierarch. │    │ • Recursive │                      │
│   └─────────────┘    └─────────────┘    └─────────────┘                      │
│                                                                                │
│   Indexer orchestrates: Chunk → Embed → Store → Retrieve                      │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ CHUNKING STRATEGIES                                                            │
│                                                                                │
│   1. Fixed-Size (Baseline)                                                    │
│      - 512 character chunks with overlap                                      │
│      - Fast, predictable                                                      │
│                                                                                │
│   2. Semantic (+15-20% accuracy)                                              │
│      - Splits at paragraph/section boundaries                                 │
│      - Preserves semantic coherence                                           │
│                                                                                │
│   3. Hierarchical/HiChunk (+15-25% accuracy)                                  │
│      - Multi-level: Document → Section → Paragraph                            │
│      - Enables coarse-to-fine retrieval                                       │
│      - Optimal leaf size: ~1,800 characters (Snowflake 2025)                  │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ RETRIEVAL STRATEGIES                                                           │
│                                                                                │
│   1. Dense-Only (Fastest)                                                     │
│      - Pure vector similarity search                                          │
│      - Latency: ~10-20ms                                                      │
│                                                                                │
│   2. Hybrid (+20-30% accuracy)                                                │
│      - Dense (semantic) + Sparse (keyword)                                    │
│      - Score fusion: weighted or RRF                                          │
│      - Latency: ~20-30ms                                                      │
│                                                                                │
│   3. Recursive (+40-50% accuracy for large docs)                              │
│      - Coarse-to-fine: Parents → Children                                     │
│      - IVF-HNSW hybrid navigation                                             │
│      - Latency: ~30-50ms                                                      │
└──────────────────────────────────────────────────────────────────────────────┘
